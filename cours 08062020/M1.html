Le développement
Le Javascript :


Javascript est un langage de programation de script employé pour rendre une page web interactive, le comportement de la page et son apparence pourra varier en fonction de certains paramètre.



Le PHP

Le PHP est un langage de programmation libre,utilisé pour généré des pages web dynamiques, qui sont donc modifiés en fonction de certaines information(heure,nom de l'utilisateur,formulaire rempli par l'utilisateur etc


MySQL:

My SQL est un système de gestion de base de données,il permet d'enregistrer des informations rempli par l'utilisateur(exemple:formulaire de création de compte)pour ensuite étre ré-utilisé au moment approprié(ex:connexion à un site internet avec identifiant et mot de passe).
